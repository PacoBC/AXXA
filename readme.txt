Se realizó la estndarización del archivo csv, para que generara los archivos automaticamente al ejecutar el programa,  se realizó con la minima operación del usuario de tal manera que lo unico que hará este es colocar la ruta de la carpeta en el que se encuentra el archivo csv.

La máquina virtual con Ubuntu Server fue creada en Microsoft Azure con dirección IP 13.84.213.109 y se abrió el puerto 8080 para poderse comunicar con Apache HTTP Server.

En dockerinsitall.sh se ecuentran todos los comandos para instalar docker en la máquina virtual.

Dockerfile se encuentra los comandos para crear una imagen con Apache HTTP Server Project.

docker-commands.txt los comandos para crear un contenedor con Apache HTTP Server Project.
